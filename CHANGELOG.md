## [0.1.0] - 2024-XX-XX

### Added
- Initial release of `testmechs` Python package.
- Sharp-null hypothesis tests (CS, ARP, FSSTdd, FSSTndd, Kitagawa methods).
- CR confidence-set method using SciPy HiGHS LP solver.
- Lower-bound calculations (`lb_frac_affected`, `breakdown_defier_share`).
- ADE/ATS bounds estimation (`bounds_ade_ats`).
- TV confidence interval inversion (`ci_TV`).
- Partial density data and plotting.
- Regression adjustment support (`reg_formula`).
- Monte Carlo simulation framework.
- Paper reproduction helpers.
- Bundled empirical datasets (Baranov, Bursztyn, Kerwin).
- Complete R-Python numerical parity verification suite.
